### 投影变换

此处用于展示各类投影下的风场效果。

#### epsg:4326

::: tip
投影变换，地图投影默认为 `EPSG:4326`, 可以看到地图两极有明显变形;
:::

请参照 ol [文档](./ol.md)

#### epsg:3413 极投影

::: tip
投影变换，地图投影默认为 `EPSG:3413` 极投影;
:::

请参照 ol [文档](./ol.md)

#### ESRI:53009 moll

::: tip
投影变换，地图投影默认为 `ESRI:53009`;
:::

请参照 ol [文档](./ol.md)
